## 온라인 도서관

김영한 스프링 로드맵을 진행하면서, 동시에 개발하는 복습 겸용 사이드프로젝트입니다.

[부천시립도서관](https://www.bcl.go.kr/)과 같은 온라인에서 도서를 대출할 수 있는 사이트입니다.

## 기능

### 유저

- 도서 대출
- 도서 조회
- 도서 반납

### 관리자

- 도서 등록
- 도서 삭제
- 도서 수정


## 점진적 고도화

김영한 스프링 로드맵을 보면서, 점진적으로 프로젝트를 고도화시킬 계획입니다. 스프링 MVC1편의 프론트 컨트롤러 v5를 시작으로 개발합니다.
