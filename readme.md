## 온라인 도서관

시립 도서관 사이트에서 아이디어를 얻어 작업한 1인 프로젝트입니다.

도서 대출, 반납 기능과 관리자용 도서 및 회원 관리 기능을 제공합니다.

유저는 사이트에서 도서 대출 후 도서관에서 간단히 관리자의 확인 후 픽업할 수 있는 서비스입니다.

배포된 페이지는 아래 링크에서 확인해주세요!

[온라인 도서관](https://web-library-front.vercel.app/)

## 주요 기능

- 로그인 및 회원가입
- 도서 검색
- 도서 대출 및 반납
- 관리자 기능
  - 도서 관리
  - 유저 관리
  - 대출 관리
